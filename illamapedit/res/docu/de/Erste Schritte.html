Wie öffne ich eine Karte?

Es gibt 2 Möglichkeiten eine Karte zu öffen.

Über das Hauptmenu

Gehe auf das Hauptmenu und klicke auf Karte öffnen:

Danach wähle mit dem Dialog den Ordner, in dem deine Karten sind.

Achtung: Die Karten werden im Dialog NICHT dargestellt.

Nun kann im nächsten Dialog eine Karte ausgewählt werden.

Jetzt wird die Karte geladen. Wenn dies geschehen ist, muss folgendes gemacht werden:
Klicke im Ribbon auf das DropDown Menu und wähle den Namen der geladenen Map aus. Nun sollte automatisch zu ihr gescrollt werden.

Über den Ribbon

Diese Methode funktioniert nur wenn der Editor den Ordner mit den Karten bereits kennt.
Im Ribbon, direkt über dem DropDown aus dem letzten Bild, befindet sich eine Liste, welche bei bekanntem Karten Ordner alle verfügbaren Karten anzeigt. Man kann mit den Pfeilen auf der Seite der Liste scrollen und mit einen Klick auf einen Eintrag wird die Karte geladen.
