Starting:

Once your editor is open, you can change a few little things such as language, maps directories (where your maps are saved), and theme.
Press the button in the top in the left corner, go to 'settings', and there you can change your settings. Make sure that you activate 'show mouse position on map'.

Creating a map:

Press the button in the top in the left corner, go to 'new map', and add in the little window your desired map settings.

- Width and height is the size of the map.
- X position is the X starting position of your map.
- Y position is the Y starting position of your map.
- Map level is the starting level of your map; level 0 is usually the surface.
- Name is the name of your map.

Buttons/menus:

Paste, copy, cut
With these tools you can copy (cut) a map and paste it in another map.
If you want to do this, you have to select the area you want to copy (cut) with the selection brush.
Select your area and press the button 'copy' (cut). Once you have done that, paste it at your desired map/location.

Tiles, Items, Empty tiles
With these buttons you can hide/show the tiles, items and empty tiles (question marks) on your map.

Grid, sounds, warps, item data, annotation, obstacle
These buttons show you the grid, sounds, warp points, item data, annotations and obstacles of your map.

Zoom
That's obvious I guess and doesn't need any explanation.

Tools:

Tile brush
With this tool, you can add tiles on your map. Make sure that you place one tile on each position.

Tile eraser
With this tool you can delete tiles from your map.

Selection brush
With this tool you can select an area on your map (mainly used for copy/paste).

Item brush
With this tool you can add items on your map.

Item eraser
With this tool you can delete items from your map.

Item data
With this tool you can add data or annotations.
Use it this way:
Select 'item data' and then select the item you want to add the data to. The item you selected will be shown on the right-hand side of your editor.
Select the item on the right-hand side and then press the '+' to add the data or the flag to add an annotation.
For the item data, a little window opens where you can add the 'key' and the 'value'.
Using a book as an example, the key could be "nameEN" and the value would be "Book of the secret map techniques".
Please keep in mind that you need a key and a value in both languages.
For the annotations, a little window opens where you can add your text.
Annotations are used for tiles/items which shouldn't be deleted, e.g. city depots which have data or quest items.

Music brush
With this tool you can add/delete music on your map and also listen to the music files we currently have.

Warp brush
With this tool you can add/delete warp points on your map.
To add a warp point you have to select a target X, Y and Z (target layer) position.
For example:
Your warp point starts at 0, 0, 0 and 0, 0, 1 should warp you to the next level.
You type your target coordinates (0, 0, 1) in the target X, Y and Z (target layer) fields and then you select the field where it should start (0, 0, 0).

Your current, opened maps are shown on the left-hand side of your map editor. You can close them if you select the "X" button or hide it by removing the tick.
